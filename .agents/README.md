# .agents

Project-local MiMoCode agent configuration.

See [README.md](../README.md) for project overview and [MIMOCODE.md](../MIMOCODE.md) for MiMoCode config details.

## Conventions

- `skills/` — on-demand skills discovered by MiMoCode.
- `rules/` — behavior rules.
- `plans/` — plan mode outputs.
- `agents/` — custom agent markdown definitions.
- `settings.json` — agent defaults (linked via `.mimocode/` symlinks).
