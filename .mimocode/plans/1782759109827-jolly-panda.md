# Plan: Multi-Language Init Config Generation

## Goal
`lint-arwaky-cli init` detects ALL languages in a project and generates corresponding config files. Generated configs must be byte-identical to reference YAMLs (already guaranteed by `include_str!`).

## Current Behavior
- `detect_language()` → returns only ONE language (rust > python > javascript priority)
- `handle_init()` → generates only ONE config file

## Target Behavior
Detect multiple languages simultaneously:
- `crates/` → generate `lint_arwaky.config.rust.yaml`
- `packages/` or `modules/` or `pyproject.toml` → generate `lint_arwaky.config.python.yaml`
- `package.json` → generate `lint_arwaky.config.javascript.yaml`
- Fallback: if nothing detected → default to `["rust"]`

## Files to Modify

### 1. Protocol trait — `crates/shared/src/project-setup/contract_setup_protocol.rs`
- Add method: `fn detect_languages(&self) -> Vec<String>;`

### 2. Aggregate trait — `crates/shared/src/project-setup/contract_setup_aggregate.rs`
- Add method: `fn detect_languages(&self) -> Vec<String>;`

### 3. Processor (capabilities) — `crates/project-setup/src/capabilities_setup_processor.rs`
- Add `detect_languages()` implementation with multi-detection logic:
  ```
  let mut langs = Vec::new();
  if Path::new("crates").exists() { langs.push("rust"); }
  if Path::new("packages").exists() || Path::new("modules").exists() || Path::new("pyproject.toml").exists() { langs.push("python"); }
  if Path::new("package.json").exists() { langs.push("javascript"); }
  if langs.is_empty() { langs.push("rust"); }
  langs
  ```

### 4. Orchestrator (agent) — `crates/project-setup/src/agent_setup_orchestrator.rs`
- Add pass-through: `fn detect_languages(&self) -> Vec<String> { self.protocol.detect_languages() }`

### 5. CLI surface — `crates/cli-commands/src/surface_setup_command.rs`
- Rewrite `handle_init()` to loop over `detect_languages()` and generate config for each

### 6. TUI executor — `crates/tui/src/capabilities_lint_executor.rs`
- Rewrite `run_init()` (line 195-233) to loop over `detect_languages()`

### 7. Tests — `crates/project-setup/tests/agent_setup_orchestrator_tests.rs`

### 8. Tests — `crates/tui/tests/lint_executor_tests.rs`
